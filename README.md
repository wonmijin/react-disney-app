## 📣 주요 기능

### 1. INTRO
  - 첫 화면
  - Google OAuth를 통한 로그인/로그아웃
    
### 2. Main
  - 모달 페이지를 통한 영화 소개글
  - Hover, Button을 통한 영상 재생

### 3. Search
  - 검색페이지
  - 검색된 영화 바로 하단에서 확인 가능
